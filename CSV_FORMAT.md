# CSV 파일 형식 가이드

LeeDeli 시스템에서 사용하는 CSV 파일 형식에 대한 설명입니다.

## 프렙 관리 CSV 형식

### 파일 구조

```csv
이름,재료명,수량,보충날짜1,보충날짜2,보충날짜3,...
오이피클,오이,10,2026-01-05,2026-01-12,2026-01-19
토마토소스,토마토,5,2026-01-03,2026-01-10
```

### 필드 설명

1. **이름** (필수): 프렙의 이름
2. **재료명** (필수): 재료 관리에 등록된 재료의 정확한 이름
3. **수량** (필수): 프렙에 사용되는 재료의 수량 (소수점 가능)
4. **보충날짜1, 보충날짜2, ...** (선택): 과거 보충 날짜들 (YYYY-MM-DD 형식)

### 주의사항

- 헤더 행(첫 번째 줄)은 필수입니다
- 재료명은 반드시 "재료 관리" 페이지에 먼저 등록되어 있어야 합니다
- 보충 날짜는 YYYY-MM-DD 형식 (예: 2026-01-05)
- 보충 날짜가 2개 이상일 경우, 평균 보충 간격과 다음 보충 예상 날짜가 자동으로 계산됩니다
- 현재 버전에서는 프렙당 하나의 재료만 지원합니다 (추후 확장 가능)

### 예시

```csv
이름,재료명,수량,보충날짜1,보충날짜2,보충날짜3
오이피클,오이,10,2025-12-20,2025-12-27,2026-01-03
토마토소스,토마토,5,2025-12-22,2025-12-29
양파절임,양파,8,2025-12-25
```

## 재료 관리 CSV 형식

### 파일 구조

```csv
이름,가격,구매단위
오이,5000,1
토마토,8000,2
양파,3000,1
```

### 필드 설명

1. **이름** (필수): 재료의 이름
2. **가격** (필수): 구매 가격 (원 단위, 소수점 가능)
3. **구매단위** (필수): 구매하는 단위 (예: 1kg, 1개 등의 숫자만 입력)

### 계산 방식

- **단위 가격** = 가격 / 구매단위
- 예: 오이를 5,000원에 1kg 구매 → 단위 가격 = 5,000원/kg
- 예: 토마토를 8,000원에 2kg 구매 → 단위 가격 = 4,000원/kg

### 주의사항

- 헤더 행(첫 번째 줄)은 필수입니다
- 가격과 구매단위는 숫자만 입력
- 구매단위는 0보다 커야 합니다
- 단위 가격은 자동으로 계산되므로 입력하지 않습니다

### 예시

```csv
이름,가격,구매단위
오이,5000,1
토마토,8000,2
양파,3000,1
상추,12000,1.5
```

## CSV 파일 생성 방법

### Excel에서

1. 위 형식대로 데이터 입력
2. "다른 이름으로 저장" 선택
3. 파일 형식을 "CSV UTF-8(쉼표로 분리)(\*.csv)" 선택
4. 저장

### Google Sheets에서

1. 위 형식대로 데이터 입력
2. "파일" > "다운로드" > "쉼표로 구분된 값(.csv)" 선택

### 텍스트 에디터에서

1. 위 예시처럼 직접 입력 (쉼표로 구분)
2. UTF-8 인코딩으로 .csv 확장자로 저장

## 업로드 방법

### 프렙 관리 CSV 업로드

1. "프렙 관리" 페이지로 이동
2. "CSV 업로드" 버튼 클릭
3. 준비한 CSV 파일 선택
4. 자동으로 데이터가 추가됩니다

### 재료 관리 CSV 업로드

1. "재료 관리" 페이지로 이동
2. "CSV 업로드" 버튼 클릭
3. 준비한 CSV 파일 선택
4. 자동으로 데이터가 추가됩니다

## 문제 해결

### "재료를 찾을 수 없습니다" 오류

- 프렙 CSV 업로드 시 재료명이 정확하지 않은 경우 발생
- 해결: "재료 관리"에서 해당 재료를 먼저 등록하거나, CSV의 재료명을 정확히 수정

### "CSV 파일 형식이 올바르지 않습니다" 오류

- 헤더가 없거나 데이터가 없는 경우 발생
- 해결: 첫 번째 줄에 헤더가 있고, 두 번째 줄부터 데이터가 있는지 확인

### 한글이 깨지는 경우

- 인코딩 문제
- 해결: CSV 파일을 UTF-8 인코딩으로 저장

### 날짜 형식 오류

- 보충 날짜가 YYYY-MM-DD 형식이 아닌 경우
- 해결: 날짜를 2026-01-05 형식으로 수정 (하이픈 사용)
